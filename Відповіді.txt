1. Git - це набір консольних утиліт, які відстежують і фіксують зміни в файлах. З його допомогою можна відкотитися на більш стару версію вашого проекту, порівнювати,  аналізувати, зливати зміни і багато іншого.
2. Файли можуть перебувати у трьох статусах: збережений у коміті (commited), змінений (modified) та індексований (staged).
3. git add  додає необхідний файл до репозиторію
4. git status показує поточний статус репозиторію і зміни в ньому
6. git commit -m зберігає поточний стан сховища
5. git init Ініціалізація Git у папці проекту
7. Необхідно вказати які зміни були зроблені 
8. Командою git log
9. Дозволяє переміщатися між гілками, створеними командою git branch
